# Apps Directory

This directory contains built-in applications for Retro Web OS.

### üìù Notepad (`notepad.js`)

A Windows-style text editor with:

- File menu (New, Open, Save, Save As)

- Format menu (Word Wrap, Font)
- Status bar with line/column display

### ‚å®Ô∏è Terminal (`terminal.js`)

A command-line interface with:

- Command history (up/down arrows)
- Tab completion
- Built-in commands:
  - `help` - Show available commands
  - `clear`/`cls` - Clear screen
  - `dir`/`ls` - List directory contents
  - `cd` - Change directory
  - `pwd` - Print working directory
  - `mkdir` - Create directory
  - `rmdir` - Remove directory
  - `cat`/`type` - Display file contents
  - `echo` - Print text
  - `touch` - Create empty file
  - `rm`/`del` - Delete file
  - `date` - Show current date/time
  - `whoami` - Show current user
  - `ver` - Show system version
  - `exit` - Close terminal
- Full VFS integration

🎮 How to Test

Interactive Testing Guide

  1. Click the Start button → Start menu should slide in
  2. Click "File Explorer" in start menu → Window opens, taskbar button appears
  3. Double-click "My Computer" on desktop → Opens My Computer window
  4. Click window's minimize button → Window disappears, taskbar button stays
  5. Click taskbar button → Window restores
  6. Click window's maximize button → Window fills screen (minus taskbar)
  7. Click maximize again → Window restores to original size
  8. Drag window title bar → Window moves
  9. Open multiple windows → Each gets taskbar button
  10. Watch the clock → Updates every second

🔄 System Integration

All components communicate via the kernel event bus:

  • Start Menu → Creates windows → Window Manager
  • Window Manager → Emits events → Taskbar
  • Taskbar → Controls windows → Window Manager
  • Desktop → Launches apps → Window Manager

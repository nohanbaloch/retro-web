🚀 How to Test

  1. Open index.html in a modern web browser (Chrome, Firefox, Edge)
  2. You should see:
    • Black screen with green terminal text (BIOS boot messages)
    • Progress bar animation
    • Transition to blue desktop background
  3. Open browser DevTools (F12) to see console logs
  4. Watch for boot sequence messages in the console
💡 Note: The system uses ES6 modules, so you need to open it via a web server or use a browser that supports file:// protocol with modules. Some browsers may require you to enable local file access.

🧪 Testing the Boot Sequence

The boot sequence follows these stages:

  1. POWER_ON - System initialization
  2. BIOS_CHECK - Display splash screen
  3. DIAGNOSTICS - Run system checks
  4. KERNEL_LOAD - Initialize kernel
  5. SERVICE_INIT - Load system services
  6. USER_SESSION - Restore user session
  7. DESKTOP_READY - Launch desktop environment

🐛 Testing BSOD (Blue Screen of Death)

To trigger a BSOD for testing, open the browser console and run:

window.RetroWeb.kernel.panic(new Error('Test BSOD'));
